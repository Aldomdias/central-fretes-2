PATCH - Corrigir erros de importação | Base 12

Arquivo alterado:
- src/services/freteDatabaseService.js

Erros corrigidos:

1) Erro:
duplicate key value violates unique constraint "origens_unique_transportadora_cidade_canal"

Causa:
Ao importar fretes/cotações para uma origem já existente, o sistema criava um novo id de origem
e tentava inserir outra origem com a mesma transportadora + cidade + canal.

Correção:
Antes de gravar, o sistema procura a origem já existente por:
transportadora_id + cidade + canal
e reaproveita o id correto. Assim, os fretes/cotações entram na origem existente.

2) Erro:
Could not find the 'duracao_ms' column of 'frete_importacoes' in the schema cache

Causa:
A tabela frete_importacoes no Supabase não tem a coluna duracao_ms, mas o sistema tentava gravar esse campo no histórico.

Correção:
Se o Supabase reclamar da coluna duracao_ms, o sistema tenta registrar o histórico novamente sem esse campo.

Como aplicar:
Copie a pasta src de dentro deste patch para a raiz do projeto.
Não suba a pasta PATCH_CORRIGIR_ERROS_IMPORTACAO_BASE12 para dentro do GitHub.
