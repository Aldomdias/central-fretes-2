O QUE ESSE PATCH COBRE:

A) As telas continuam separadas:
- Formatação de Tabelas = cadastro/manual guiado
- Importar Template = upload de template preenchido pelo transportador

B) Formatação de Tabelas:
- Vigência automática: início = hoje e fim = +3 anos
- Mantém transportadora nova ou cadastrada
- Geração de fretes por rota + região + todas as faixas
- Exportação de planilha para preenchimento
- Reimportação da planilha preenchida
- Listas mais controladas para não poluir a tela

C) Importar Template:
- Leitura de Rotas + Fretes separados
- Botões para baixar modelos oficiais
- Leitura de cotações por CAPITAL / INTERIOR 1 / INTERIOR 2 / INTERIOR 3
- Correção para não pegar apenas parte de SP
- Conversão do template para o modelo flat do sistema

Depois de substituir:
- parar o servidor
- rodar npm install, se necessário
- rodar npm run dev ou pnpm dev
- testar primeiro Importar Template
- depois testar Formatação de Tabelas
