# Laudos de Negociacao

Os laudos visuais da simulacao sao persistidos em `tabelas_negociacao.resumo_simulacao.laudos`.

Estrutura principal:

- `laudos.executivo`
- `laudos.transportador`

Cada item guarda assunto, corpo do e-mail, laudo completo, data de geracao e `dados` para renderizacao visual.
